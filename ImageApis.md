## Amazon:
* Size: Flexible

* Different Formats:  
QL: Quality  
FM: Format  
US: Both side to be given Size  
SL: Longest size to be given size  
AC: Space on left(US), No Space(SL)  
UY: On Y axis  
UX: On X Axis
